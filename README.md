# NAME

Test::LWP::Capture - Mock LWP requests using captured data

# VERSION

Version 0.01

# SYNOPSIS

    use Test::More;
    use Test::LWP::Capture file => '/path/to/file';

    # arbitrary code with LWP usage

# EXPORT

Nothing is exported by default.
